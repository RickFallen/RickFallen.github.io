激活函数与梯度的一些思考

发表于 2020-01-13 更新于 2021-03-02 分类于 deep learning
本文字数： 456 阅读时长 ≈ 1 分钟

常见激活函数

Sigmoid

优点 :

  1. 把函数值压缩到了[0,1]范围内
  2. have nice interpretation as a staturating "firing rate" of a neuron

缺点 :

  1. 梯度消失

    当\(x\)的值取正负无穷大时，Sigmoid函数的导数几乎为零，这导致在反向传播应用链式法则时梯度几乎为0。

  2. 输出结果不是"0中心"

  3. 指数函数计算量过大

tanh

优点 :

  1. 把函数值压缩到了[-1,1]内
  2. 输出结果是"0中心"

缺点 :

  1. 仍然会出现梯度消失

Relu

优点 :

  1. 当 \(x > 0\) 时不会出现梯度消失
  2. 计算量小，并且在实践中比sigmoid和tanh收敛速度要快很多

缺点 :

  1. 输出结果不是"0中心"
  2. 当\(x < 0\)时，会出现梯度消失
